안녕하십니까 교수님 1팀(정세벽, 김범준, 서현영)입니다.

저희는 교수님께서 구현하라고 팀프로젝트 게시판에 올리신 모든 기능을 구현하였습니다.

일단, 프로그램이 돌아가는 방식은, 제가 올린 파일인 teamp1.py 파일을 다운받고, 안에 import 된 모든 것을 install 하신 후에, 
ElasticSearch를 실행시킵니다. 그리고 teamp1.py이 저장된 폴더에서 ./teamp1.py —listen-port=8899 를 치시면
http://155.230.91.204:200XX/
(XX는 접속한 vm번호)
사이트에 제희 팀이 만든 사이트가 실행될 수 있게 됩니다.

접속하시면 Home 페이지가 나올텐데, 그 곳에 단일 URL을 입력할 수 있는 Text박스와 Analyze버튼, 그리고 파일을 제출할 수 있는 폼이 있습니다. 단일 URL을 입력하거나 파일을 입력하면 http://155.230.91.204:200XX/status 페이지로 넘어가게 됩니다. 그 곳에서는 입력을 처리하는 과정에서의 상태나 오류등을 표시할 수 있는 status text box가 존재하여 처리한 URL, 처리 성공 여부, 실패이면 그 원인 설명, 중복 URL 존재 여부 등의 사용자 편의를 위한 메시지를 출력됩니다. 여기서 Crawling에 성공한 URL들이 중복되지 않게 선별되어 ElasticSearch에 저장됩니다.

그리고 OK버튼을 누르면 http://155.230.91.204:200XX/result 페이지로 넘어가게 됩니다.
그 곳에는 웹사이트 분석 결과 표가 존재하는데, ElasticSearch에서 선별된 URL들을 불러와서 총 단어 수, 단어들, 빈도수를 URL과 함께 ElasticSearch에 저장합니다. 그리고 이 페이지에서는 아래와 같은 기능을 합니다.

- HTML table 형태로 URL들의 리스트들이 표시 되고 각 줄 마다 전체 단어수, 처리시간(초), 그리고 다음의 기능을 가지는 버튼 혹은 링크가 존재하여 클릭할 수 있다.
[1] TF-IDF 처리를 통한 상위 top 10 중요한 단어들의 리스트를 보여주는 팝업 HTML 창
[2] Cosine similarity를 사용하여 판단한 top 3 웹 URL의 리스트를 보여주는 팝업 HTML 창

TF-IDF 버튼과, Cosine-Similarity버튼이 있습니다. 그리고 Home으로 돌아가는 버튼을 하나 더 만들어 두었습니다.

TF-IDF 버튼을 누르면 http://155.230.91.204:200XX/tfidf 페이지가 팝업으로 뜨게 됩니다. 
이 곳에는 TF-IDF 처리를 통한 상위 top 10 중요한 단어들의 리스트가 보여지는데, 아까 /result 페이지에서 저장해 둔 URL과 단어를 ElasticSearch에서 불러와서 계산합니다.

Cosine-Similarity 버튼을 누르면 http://155.230.91.204:200XX/cosine 페이지가 팝업으로 뜨게 됩니다.
이 곳에는 Cosine-Similarity 를 사용하여 판단한 top 3 웹 URL의 리스트가 보여지는데, 아까 /result 페이지에서 저장해 둔 URL과 단어를 ElasticSearch에서 불러와서 계산합니다.

Go Home버튼을 누르면 http://155.230.91.204:200XX/ 페이지인 Home 페이지로 돌아가게됩니다.
